
3. Validación en `CatalogController::upload()`:
   ```php

   $maxItems = $tenant->plan->max_catalog_items ?? 500;

       return response()->json([
           'success' => false,
           'message' => "Tu plan permite máximo {$maxItems} items. Upgrade tu plan para subir más productos."
   ```

4. Nuevo endpoint: `GET /api/plans` - listar planes disponibles

Crear un wizard de 5 pasos para nuevos usuarios:

```
Paso 1: Información del Negocio

  - Descripción (textarea)
  - Horarios de atención (json: {monday: {open, close}, ...})

Paso 2: Conectar WhatsApp
  - Mostrar QR para escanear
  - Estado de conexión
  - Botón para reconectar

Paso 3: Configurar Catálogo
  - Elegir tipo (productos/servicios/ambos)
  - Subir archivo (ya implementado en CatalogView)
  - Ver items subidos

Paso 4: Personalizar Chatbot
  - Custom prompt (instrucciones adicionales)
```

**Endpoints necesarios:**
- `GET /api/tenant/onboarding` - obtener estado actual
- `PUT /api/tenant/onboarding` - guardar progreso (cada paso)
